# Pennywise App

Clone the repo:

```
git clone https://github.com/ncarchedi/pennywise-app.git
```

Install packages:

```
yarn install
```

Run the production version:

```
cd pennywise-app
yarn start
```

Run the development version:

```
cd pennywise-app
git checkout dev
yarn start
```

Run tests:

```
yarn test
```

Link to the backend: https://github.com/BGordts/pennywise-app-backend.git
